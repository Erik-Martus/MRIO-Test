# Front End Developer Exercise

---

## Get Started

Install dependencies:

```sh
npm install
```

Run application:

```sh
npm start
```

---

## Technologies

- MUI
  - React community library for Material UI
- React
- Typescript
- Node.js
- HTML/CSS
- SASS
- Redux
- Jest
- Webpack
  - Code compilation and bundling
- ESLint & Prettier
  - Code formatting and linting
